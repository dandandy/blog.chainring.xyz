Simple Tips for Happy Bashing

Bash (Bourne Again Shell) is an incredibly common shell and scripting language in which any developer working on Linux and macOS based systems will inevitably come across. Because of this, Bash scripting skills are a formidible tool to have in your belt. However, Bash scripting is notoriously difficult due to its extensive nature and large feature set, which often leads to developers writing poor scripts. In this post, we will explore you can write better Bash scripts with the help of The Set Builtin, Trapping Functions, Shellcheck, and Idempotency.
